# Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Project Overview

This project was completed as part of a Data Analyst Internship (Task 5). The objective was to perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python to identify patterns, trends, relationships, and insights through statistical summaries and visualizations.

---

## 🛠 Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset

The Titanic dataset contains passenger information such as:

- Passenger ID
- Survival Status
- Passenger Class
- Name
- Gender
- Age
- Fare
- Cabin
- Embarked Port

---

## 📊 Exploratory Data Analysis Performed

- Data Loading
- Dataset Overview
- Statistical Summary
- Missing Value Analysis
- Value Counts
- Survival Count Plot
- Gender Distribution
- Age Histogram
- Age Boxplot
- Survival by Gender
- Survival by Passenger Class
- Fare Distribution
- Correlation Heatmap
- Pairplot
- Scatter Plot

---

## 📈 Key Findings

- The dataset contains 891 passenger records.
- Missing values were identified in Age, Cabin, and Embarked.
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had better survival chances.
- Most passengers were between 20–40 years old.
- Ticket fares were positively associated with passenger class.
- EDA helped uncover meaningful trends and relationships within the dataset.
